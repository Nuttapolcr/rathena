    CharmScript: |
      bonus bStr,1;

    Name: Test Charm Upgrade
    Type: Charm_Upgrade
    Refineable: true
    CharmUpgradeMaxEffect: 1
    CharmScript: |
      .@r = getcharmhrefine();
      bonus bLuk,.@r;

  - Id: 100103
    AegisName: Test_Charm_Upgrade_II
    Name: Test Charm Upgrade
    Type: Charm_Upgrade
    Refineable: true
    CharmUpgradeMaxEffect: 5 # กำหนดค่า จำนวน charm ที่จะส่งผล โบนัส
    CharmScript: |
      .@r = getcharmrefine();
      bonus bDex,.@r;

-- script เพิ่มเติม --

getcharmrefine(); ไว้ดึงระบบตีบวกของชาม
getcharmgrade(); ไว้ดึงเกรดของชาม

getcharmhrefine(); ไว้ดึงค่าตีบวกสูงสุดของ charm เลขนั้นๆ
getcharmhgrade(); ไว้ดึงค่าเกรดสูงสุดของ charm เลขนั้นๆ

charmamount(); เอาไว้ดึงจำนวนของ charm เลขนั้นๆ

-- หมายเหตุสำคัญ --

CharmMaxEffect จะสามารถใช้ได้เพียงกับ "Charm" ธรรมดาเท่านั้น

หากไม่กำหนด CharmUpgradeMaxEffect จะมีค่าตั้งต้นเท่ากับ "1"

-- เพิ่มเติม --
หากต้องการกำหนด ให้ Charm Upgrade มีผลแค่ 1 ชื่น/itemid

ให้กำหนด CharmUpgradeMaxEffect มีค่าเป็น 1 / หรือไม่ต้องกำหนด
และใช้งาน 
- getcharmhrefine();
หรือ
- getcharmhgrade();

getcharmhcomrefine(<item id>);
- ใช้ดึงตีบวกสูงสุดของชามเลขนั้นๆ

getcharmhcomgrade(<item id>);
- ใช้ดึงเกรดสูงสุดของชามเลขนั้นๆ
